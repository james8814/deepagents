## 发现的问题

### 代码生成测试失败分析

**测试**: Python 斐波那契函数生成
**预期**: 输出纯代码
**实际**: Agent 创建了 `/fibonacci.py` 文件并返回文件路径

**评估**: 这不是 bug，而是 Agent 主动行为。Agent 选择将代码写入文件而非直接输出，这是合理的文件操作行为。在实际使用场景中，这种做法可能更实用。

**建议**: 如需纯代码输出，可在 prompt 中明确要求"直接输出代码，不要写入文件"。
